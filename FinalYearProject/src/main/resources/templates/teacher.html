Quick Guide

As a Teacher, you can:

  • Add Questions - Create new questions with subject, CO, marks, and cognitive level
  • My Questions - View and manage questions you've created
  • Search Questions - Filter by subject code, name, CO, or cognitive level
  • Generate Papers - Automatically create balanced question papers
  • Submit for Approval - Send generated papers to supervisor for review
  • Download Papers - Download approved papers as PDF

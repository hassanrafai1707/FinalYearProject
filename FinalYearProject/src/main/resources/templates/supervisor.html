Supervisor Guide

As a Supervisor, you can:

  • Browse all questions in the system
  • Search and filter questions by various criteria
  • View all question papers submitted by teachers
  • Review pending papers and add feedback comments
  • Approve well-prepared papers
  • Reject papers that need revisions with constructive feedback
  • Download approved papers as PDF
